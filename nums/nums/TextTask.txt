Задача набора суммы из произвольных чисел
Напишите, пожалуйста, программу для набора суммы из множества элементов. На вход программа получает число которое надо набрать и текстовый файл, который содержит по одному числу в строке. 
Номер строки будем считать индексом. Нужно вывести (на экран или в файл) индексы, сложив данные которых получим искомую сумму. Одну строку можно использовать только один раз.

Например: Нужно набрать 555,55

№ строки в файле
число
1
10,50
2
550,00
3
1000
4
5,55
...
...
99999999
541,30
Программа должна вывести: 2,4

Требования:
ОС: Windows
ЯП: C#/VB.NET
Без использования рекурсии
